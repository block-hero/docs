## Create your first project

Follow these steps to set up your first Block Hero project.

## Step 1: Create a new project

1. Click **New Project** in the dashboard
2. Enter a project name
3. Choose your project type
4. Click **Create**

## Step 2: Configure basic settings

1. Go to **Project Settings**
2. Set your project preferences
3. Configure your first integration
4. Save your changes

## Step 3: Invite team members

1. Go to **Team** → **Members**
2. Click **Invite Member**
3. Enter their email address
4. Select their role and permissions
5. Send the invitation

## Common questions

<AccordionGroup>
  <Accordion title="How do I change my project name?">

  </Accordion>
  <Accordion title="Can I have multiple projects?">
